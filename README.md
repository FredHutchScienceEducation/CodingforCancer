# Coding for Cancer
Instructional material for Coding for Cancer. 

The [Coding for Cancer Program](https://www.fredhutch.org/en/education-training/high-school-students/coding-for-cancer.html) at the Fred Hutch Cancer Center (Fred Hutch) is a virtual summer program that provides 11th and 12th-grade high school students with introductory experiences in computational biology and cancer research. 

See our Ten Simple Rules article in PLOS Computational Biology. 

## Contents

- `coding_help_sheets/` - supplemental documents for students, syntax and function references.
- `daily_warmup/` - example questions to start each class session
- `datasets/` - datasets used for data exploration during lessons
- `final_project_presentations/` - two example student presentations completed during the 2024
program.
- `identify_artifact/` - example identify artifact
- `rmd_lessons/` - instructional material on R markdown file format
- `swirl_lessons/` - instructional material on [swirl](https://swirlstats.com/)
