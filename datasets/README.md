# Datasets

Example datasets used for data exploration activities

- [palmerpenguins](https://allisonhorst.github.io/palmerpenguins/) - measurements of 344 penguins collected from 3 islands in the Palmer Archipelago, Antarctica.
- [tidytuesday](https://github.com/rfordatascience/tidytuesday) - community curated datasets for data exploration and visualization. Updated weekly.
- `pokemon.csv` - game data of 1010 Pokemon. Sourced from [Bulbapedia](https://bulbapedia.bulbagarden.net/wiki/Main_Page)
- `stardew_valley_crops.csv` - game data of crops grown in Stardew Valley. Source from [Stardew Valley wiki](https://stardewvalleywiki.com/Stardew_Valley_Wiki).
- `clinical.csv` - subset of bladder cancer patients from the [NCI Cancer Genome Atlas](https://www.cancer.gov/ccg/research/genome-sequencing/tcga).
