Vectors

Recall from last lesson, we assign single values inside a variable like this:

my_age <- 16


If we want store multiple objects, we assign them to an object called a vector.

To do this in R, we put the values in this c() function. The c() function is used to combine values.

ages <- c(50,55,60,65)
ages


Here’s a more detailed look into each part of the code to make a vector:


We can use the c() function to add more values is an existing vector. 
more_ages <- c(ages, 70)
more_ages

Indexing

Each element in a vector has an associated index, or position, that we can use as a reference. The index tells us the position of the element in the vector.

Take the following vector as an example: 
organs <- c("lungs", "heart", "kidney", "brain", "stomach", "liver", "spleen")

If we want to access the 3rd value in organs, we use square brackets [] with the vector like this: organs[3]. 
organs[3]

In the code chunk below, extract the 6th item in organs.

Use square brackets [] with the number 6 inside: organs[6]

Subsetting

Recall that the organs vector is:

organs <- c("lungs", "heart", "kidney", "brain", "stomach", "liver", "spleen")


We can also use indexing to extract multiple values from the vector. This is called subsetting a vector.

To get the first through third elements use the : symbol like this: organs[c(1:3)].


To get the only first and third, we need to first make a vector with 1 and 3 or c(1,3) and then put the vector in square brackets [].

index <- c(1,3)
organs[index]
index <- c(1,3)
organs[index]

Alternatively, it can be written in one line: 
organs[c(1,3)]
