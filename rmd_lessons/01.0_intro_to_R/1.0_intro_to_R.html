Data types

In the previous example, we assigned apples with the value 10. 10 a type of data called an integer.

apples <- 10

The most common data types we will work with are:

  • Character, also called strings: Contains a series of letters or symbols enclosed in " or '. Examples: "Hello!", "coding4%", "good morning 52"
  • Numerics: Contains any type of number data. Examples: 4.1503, 10.502, -205.2
  • Integers: Contains number data, but do not have decimal points. Examples: 5, -19, 201
  • Logicals, also called booleans: Contains only TRUE or FALSE. It can be also represented as T or F, respectively.


This is an example of a character or string data type:

city = "Seattle"
